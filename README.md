# Node ES6 Boilerplate

Boilerplate for Node.js projects.

## Installation

1. Clone repo    
`git clone git@github.com:samyabouseda/node-es6-boilerplate`    

2. Change directory    
`cd <dir_name>`    

3. Remove git folder    
*nix: `rm -rf ./.git`    
win: `RD /S /Q .git`    

4. Edit **package.json**    
Change `name, version, description, repository, author, bugs, homepage`    

5. Install packages    
`npm install`    

6. Update packages    
`npm update`    


## Scripts

1. `npm start`    
Transpile ES6 and start node server.

2. `npm test`
Run tests.
